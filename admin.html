Yedekleme Dosyaları 📂

Dış zamanlanmış görev (Cron-job.org) üzerinden her gün tam saat 06:00'da (TSİ) otomatik yedekleme tetiklenmekte, GitHub Actions aracılığıyla yedek alınarak Cloudflare R2 ve Google Drive hesaplarınıza yüklenmektedir. Buradan manuel yedekleme başlatabilir ve mevcut yedekleri indirebilirsiniz. i

ℹ️ Eski Çalışma Prensibi & Güncelleme Notu:
Eskiden yedekleme işlemleri doğrudan GitHub Actions'ın kendi iç zamanlayıcısı (scheduler) ile tetikleniyordu. Ancak GitHub'ın ortak sunucu kuyruk yoğunluğuna bağlı olarak 06:00'da başlaması gereken yedekler bazen 06:30 veya 06:50 gibi gecikmeli çalışabiliyordu. Yedeklerin tam saat 06:00:00'da kararlı şekilde alınabilmesi için harici zamanlayıcı (Cron-job.org) üzerinden API tetiklemeli yeni sisteme geçilmiştir.
